  Step 2: Rearrange your windows
 

With the window to move and resize frontmost call Grid's panel with your keyboard shortcut.

The colored area shows which fraction of the screen your window will occupy after moving and resizing.

Now use the keyboard to set the new position and size of the window:

  • Move: Use the arrow keys to move the colored area.
  • Resize: Hold the Shift key and use the arrow keys to change the size of the area.
  • Change the grid: Hold the Command key and use the arrow keys to change the grid parting horizontally and vertically.
  • Apply the changes: Press Return.
  • Cancel: Press Escape to close the Grid panel without changing anything.

If you have multiple monitors, you can switch between them with Command-Tilde and Command-Shift-Tilde (same as to switch between windows in any app). On a German or French keyboard use Command-< and Command-Shift-<.

Happy gridding!
