Installing and Updating

Copy Grid into the Applications folder or any other directory you have access to. Use System Preferences > Accounts > My Account > Startup Items to add Grid to your startup items. If you do so, don't select Hide because otherwise Grid won't be recognized as a 'faceless' background application by the system and therefore an icon in the Dock could appear.

To update an existing version of Grid, simply copy the application package from the disk image to your Applications folder, replacing the old version.

If you installed Grid from the Mac App Store, please use the App Store application for updating this software.


Uninstalling

To remove (uninstall) Grid completely from your Mac, simply move the Grid application package to the trash. If you installed Grid from the Mac App Store please deinstall it using the App Store application.
